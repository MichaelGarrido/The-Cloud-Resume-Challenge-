## Cloud Resume Project

Michael built a serverless cloud resume and portfolio using AWS S3, CloudFront, Route 53, API Gateway, Lambda, DynamoDB, ACM, CloudWatch, SNS, PagerDuty, Terraform, GitHub Actions, pytest, and Cypress.

The project includes:

- Static frontend hosted on S3 and served through CloudFront.
- Custom domain managed with Route 53.
- HTTPS certificate managed with ACM.
- Python Lambda API for visitor tracking.
- DynamoDB table for persistent visitor count.
- API Gateway endpoint for frontend/backend communication.
- Terraform infrastructure as code.
- Remote Terraform state in S3 with DynamoDB locking.
- GitHub Actions CI/CD pipeline.
- Backend unit tests with pytest.
- End-to-end browser tests with Cypress.
- CloudFront cache invalidation after deployment.
- CloudWatch alarms, SNS notifications, and PagerDuty alerting.
